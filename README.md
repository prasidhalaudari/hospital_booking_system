# Hospital Booking System

This is a desktop application developed as part of my academic project to manage hospital appointment bookings.

The system allows staff to register patients, view patient information, and manage appointment scheduling in an organised way. It was built using C# Windows Forms with SQL Server for database storage.

---

## Main Features

- Patient registration and record management  
- Appointment booking and scheduling  
- SQL Server database connectivity  
- Simple and easy-to-use WinForms interface  

---

## Project Structure

The application follows a 3-tier architecture:

- **UI Layer** – Handles the Windows Forms interface  
- **Business Logic Layer** – Contains the main application logic and validation  
- **Data Access Layer** – Manages SQL queries and database operations  

---

## Technologies Used

- C#  
- .NET Windows Forms  
- SQL Server  
- Visual Studio
